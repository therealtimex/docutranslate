使用教程

视频教程可以在b站搜索 docutranslate 获取

欢迎使用 DocuTranslate！请按照以下步骤完成文档翻译：

  1. 选择工作流

    首先，在配置面板顶部选择您需要的翻译流程。例如，如果您要翻译PDF或Word文档，请选择“转Markdown再翻译”。如果要翻译一个.txt文件，请选择“纯文本翻译”。

  2. 配置参数

    根据您选择的工作流，完成相应的配置：

    • 解析配置 (仅在“转Markdown再翻译”工作流下显示):
      • 选择一个解析引擎。推荐使用 minerU 进行云端解析（支持PDF/Word/PPT/jpg等），如果需要本地解析，可以选择 Docling。
      • 如果使用 minerU，请填入您的 Mineru Token。
      • 如果输入文件是markdown文件，该部分可以不进行设置
    • 翻译模型:
      • 选择您的AI服务提供商平台，或选择 自定义接口。
      • 填入您的 API Key 和 模型 ID（例如 glm-4-flash、deepseek-chat）。
      • API-KEY和模型ID参见对应平台文档。
      • 建议使用非推理模型进行翻译。
    • 翻译配置:
      • 选择您希望翻译的 目标语言。
      • 可以填写自定义Prompt，指导AI进行特定翻译，例如“人名保持原文不翻译”。
    • 高级参数:
      • 通常保持默认即可。
      • 并发数指的是每个任务允许多少个并发AI翻译请求，需参照平台模型速率限制进行设置
  3. 上传文件

    在右侧的任务列表中，点击或拖拽您的文档到文件上传区域。

  4. 开始翻译

    文件选择成功后，点击任务卡片右下角的 开始翻译 按钮。系统将开始处理任务，您可以在日志区域查看实时进度。

  5. 查看与下载

    翻译完成后，任务卡片下方会出现操作按钮（根据文件类型动态显示）：

    • 预览: 在右侧滑出的面板中进行原文和译文的对照预览。
    • 下载 PDF: 将译文直接保存为PDF格式。
    • 下载其他格式: 下载HTML、Markdown或TXT等格式的译文。
提示: 所有配置都会自动保存在您的浏览器本地，方便下次使用。
